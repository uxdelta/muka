# Muka Design System

A comprehensive design system that provides a consistent and scalable foundation for building user interfaces.

## Overview

Muka is a design system that combines design tokens, components, and guidelines to create a cohesive user experience across different brands and platforms.

## Key Features

- **Design Tokens**: A robust token system for colors, typography, spacing, and more
- **Component Library**: Reusable UI components built with design tokens
- **Theme Support**: Light and dark mode support with brand-specific variations
- **Responsive Design**: Built-in support for mobile, tablet, and desktop layouts
